📚 MyLibrary – Library Management System

A modern Library Management System built with React that helps manage books, members, and transactions efficiently with real-time data handling and analytics.

🚧 This is an ongoing project — new features and improvements are actively being added.

🚀 Live Demo

👉 https://mylibrary-lovat.vercel.app/

📌 Features
📖 Manage books (Add, Edit, Delete)
👤 Manage members (CRUD operations)
🔄 Book issue, return, and renewal system
⏰ Display overdue books
🔥 Display most read books
🧾 Track borrowing history (issue, return, renew)
📊 Visual analytics for borrow & return data
⚡ Real-time data updates using Firebase
🧩 Interactive data tables using MUI Data Grid
📱 Responsive and clean UI

🛠️ Tech Stack
Frontend: React JS 
Database: Firebase Realtime Database 
UI Library: Material UI (MUI Data Grid) 
Charts: Recharts 
Build Tool: Vite
